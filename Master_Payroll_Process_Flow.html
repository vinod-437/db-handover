Functional Breakdown

1. Employee Setup

openappointments empsalaryregister uspgetmastersalarystructure
  • Manages onboarding credentials, assigned organizational units, and master salary limits.
  • Dynamic construction of salary structures handling default UI templates.

2. Attendance Logging

uspsavebulkattendance_business cmsdownloadedwages
  • Receives bulk biometric/timesheet uploads.
  • Verifies temporary staging data and calculates base loss-of-pay and billable days moving into persistent tables.

3. Salary Processing

uspmonthwisebilling tbl_monthlysalary
  • Aggegates processed attendances with ad-hoc allowances and ledger deductions.
  • Centralizes ESI, PF, VPF, LWF, and Tax algorithms into a monolithic monthly array.

4. Liability Checking

uspmonthwiseiablityreport
  • Acts as a polyfill function assessing client funds, arrears, and disbursement totals.
  • Generates JSON arrays mapping internal employer liability to physical cash-flow.

5. Slip Generation

uspgeneratesalaryslip uspgetreportfields
  • Interfaces cleanly with the frontend allowing employees to download PDF breakdowns.
  • Employs dynamic column masking based on specific customer ID override patterns.

6. Final Bank Payout

uspemployerbankpayout
  • Validates adequate customer balance against finalized salary lists.
  • Packages flat-files in vendor-specific formats (HDFC vs HSBC) marking the database ledger as 'Transferred'.
